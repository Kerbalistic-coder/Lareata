How to Contribute

We could greatly use your help! We are still looking for contributors to Lareata. If you want to sign up, check here: Lareata Signups. However, please be advised to be polite on all of our Github. If you are impolite, we have the power to ban you and remove your ability to access Lareata for up to six months. That said however, we are looking forward to your contributions!

Warning! Please be polite or you will be banned! NO bad word usage is permitted either.

Here are some useful links:

  • Lareata Signups: Of course, you have to sign up on Github before you can contribute! You can get to the rest of the Lareata discussions fromhere to.
  • Lareata Github Home: Here is the main page for Lareata's code base.
  • Scratch Forum: While the Scratch forum is primarily used by the original employees, you might find use in it. If you creata a new Scratch account, name it the same as your Github account!
  • Godot Home: Here you can download Godot to help work on game code. Get the 3.x.x versions, we are not compatible with 4.0-alpha yet.
